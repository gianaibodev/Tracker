# BPO CSR Tracker

A mobile-first, responsive performance and time tracker for Customer Service Representatives.

## Features
- Clock in/out
- Break tracking (WC, Meal, etc.)
- Call logging with outcomes
- Deposit tracking
- Admin dashboard with real-time stats
- Detailed user performance summaries

## Tech Stack
- Next.js 15 (App Router)
- Supabase (Auth & Database)
- Tailwind CSS
- Lucide Icons
